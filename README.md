# B773

B773 is a Discord bot for aviation.

## Usage

To run a command, @mention the bot, followed by the command and its parameters.
Commands:

- `METAR <icao>`: Retrieve METAR information for a given airport
- `TAF <icao>`: Retrieve TAF information for a given airport  

*METAR and TAF information is obtained from [aviationweather.gov](https://www.aviationweather.gov/dataserver)*

## Installation

1. Install [luvit](https://luvit.io/install.html)
2. Clone this repository
3. Run `lit install` to install dependencies
4. Copy `config.toml.sample` to `config.toml` and add your bot's token
5. Start the bot: `luvit main.lua`
